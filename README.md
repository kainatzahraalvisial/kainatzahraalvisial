## 🛡️ Cybersecurity — Active Research (FYP)

### **AutoPolicy: Anomaly Detection & Automated Zero-Trust**
*Automating security policy generation for resource-constrained IoT networks*

**Hybrid ML & GNNs**: Modeling device relationships as graphs to identify behavioral abnormalities.

**Transformer Policy Synthesis**: Generating precise JSON-based zero-trust policies from traffic patterns.

**Real-time Enforcement**: Using **Open Policy Agent (OPA)** for dynamic rule deployment.

**Traffic Analysis**: Real-time capture and feature extraction utilizing **Zeek**.

**Interactive Dashboard**: Role-based monitoring platform built with **Flutter**.

**Validation**: Evaluated against industry-standard datasets like **CIC-IoT2023** and **RT-IoT2022**.
